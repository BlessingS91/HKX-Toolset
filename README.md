# HKX-Duplicator

HKX-Duplicator is a small Python script compiled into a .exe that duplicates .hkx animation files for MCO or BFCO movesets. It automatically creates 10 copies of the selected file, numbered 1–10.

**Notes:**
*Will only work on mco / bfco attack or power attack files, will exit if tried to use on anything else.
*Must only have a single .hkx file within the folder or it will exit out to prevent accidental overwrites.

**Features:**
Duplicates MCO or BFCO attacks or power attacks.
Automatically names copies with numbers 1–10 to fill every slot.
Remembers the last folder you used for convenience.
Provides warning popups for invalid files or unsafe conditions.

**Usage:**
Run HKX Duplicator.exe
Select the .hkx file you want to duplicate.
The script will create 10 copies in the same folder.
Follow popup warnings if any issues are detected.
